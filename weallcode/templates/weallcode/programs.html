Courses

Week-long summer experiences offering students the chance to learn in-depth skills like project management, quality assurance, and social skills along with code.

HTML 101

Choose Your Own Adventure

Prerequisities
  • None
Pathways
  • Front-end Development

Think HTML and CSS are something you'd find in alphabet soup? Ever wonder how the web can deliver everything from Minecraft maps to YouTube Videos? Eager to learn and ready to build?

By the end of the session, students will understand how the language HTML, CSS, and Javascript work together to build the web. Students will develop an understanding of how HTML is used to define the structure of web pages. In keeping with CoderDojoChi's philosophy of learning-by-doing, students will use their knowledge to hack together their own "Choose Your Own Adventure" game.

CSS 101

Designing Your Adventure

Prerequisities
  • HTML 101
Pathways
  • Front-end Development
  • Design

Have you mastered HTML? Are you ready to level up? Do you want to add more of your personality to your website?

By the end of the session, students will understand how CSS enhances the look of the web. Students will develop an understanding of how to control the design and layout of web pages. In keeping with CoderDojoChi's philosophy of learning-by-doing, students will use their knowledge to style their own profile page.

JS 101

Drawing with Javascript

Prerequisities
  • None
Pathways
  • Front-end Development

Ever wanted to start building your own game but didn't know where to start? Ever thought it would be cool to have a computer draw a picture or create an animation for you?

By the end of the session, students will have the understanding of Canvas using the Javascript programming language. Students will learn how to break down tasks and objects to understand how they are built.

PY 101

Robotics with Python

Prerequisities
  • None
Pathways
  • Back-end Development

Have you always wanted to try robotics? Do you think it would be cool to learn the coding to make actual robots move and groove?

By the end of the session, students will have a basic understanding of the Python programming language and how to use it to "drive" their robots. Students and guardians can even learn together so you can talk about your robot adventures for days to come!
